# Crowd-Funding-App

## Description

Crowd-Funding-App is a Java console application that models a crowdfunding platform where creators launch campaigns, backers contribute funds, VIP users earn rewards, and campaign outcomes are processed based on funding goals.

The project demonstrates Object-Oriented Programming (OOP) concepts such as:
- Inheritance
- Encapsulation
- Polymorphism
- Class Relationships

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Eclipse IDE

## Features

- Creator and Backer Management
- Campaign Creation and Tracking
- Flexible Funding Campaigns
- VIP Backer Rewards
- Cashback and Points System
- Funding Goal Monitoring
- Campaign Settlement Processing
- Wallet Balance Management

## Sample Output

- Campaign progress tracking
- Funding goal evaluation
- VIP reward calculation
- Wallet balance updates
- Campaign settlement status
