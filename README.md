Crowd-Funding-App is a Java console application that models a crowdfunding platform where creators launch campaigns, backers contribute funds, VIP users earn rewards, and campaign outcomes are processed based on funding goals. The project demonstrates OOP concepts such as inheritance, encapsulation, polymorphism, and class relationships.


Technologies
Java
Object-Oriented Programming (OOP)
Eclipse IDE


Features
Creator and Backer Management
Campaign Creation
Flexible Funding Campaigns
VIP Backer Rewards
Cashback & Points System
Funding Goal Tracking
Campaign Settlement Processing
Wallet Balance Management
